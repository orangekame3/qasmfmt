# Installation

## From crates.io (Recommended)

```bash
cargo install qasmfmt
```

## From Source

```bash
git clone https://github.com/orangekame3/qasmfmt
cd qasmfmt
cargo install --path .
```

## Python

```bash
pip install qasmfmt
```

## npm

```bash
npm install qasmfmt
```

## Verify Installation

```bash
qasmfmt --version
```
