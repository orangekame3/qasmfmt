# Contributing

## Development Setup

```bash
git clone https://github.com/orangekame3/qasmfmt
cd qasmfmt
cargo build
```

## Running Tests

```bash
cargo test
```

## Code Style

Format code before committing:

```bash
cargo fmt
cargo clippy
```

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make changes
4. Run tests
5. Submit PR

## Reporting Issues

Please include:
- qasmfmt version
- Input QASM code
- Expected output
- Actual output
